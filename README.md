# Prueba-MeLi
 Prueba Técnica Frontend Johan C.

## Scripts

### `npm i`

Instalará las dependencias del proyecto
### `npm start`

Inicia el Cliente y el Servidor en paralelo

Cliente: PORT 3001
Servidor: PORT 3002

### `npm start-client`

Inicia el Cliente de forma independiente

Cliente: PORT 3001

### `npm start-server`

Inicia el Servidor de forma independiente

Servidor: PORT 3002
### `npm test`

Ejecuta las pruebas de Cliente y Servidor
